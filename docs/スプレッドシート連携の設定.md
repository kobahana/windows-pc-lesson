# 学習記録をGoogleスプレッドシートに集める設定

生徒がどのパソコンで学習しても、記録が先生のGoogleスプレッドシートに自動で1行ずつ追加されるようになります。
設定は最初に1回だけ。所要時間は10分くらいです。

## しくみ

```
生徒のPC（アプリ） → Google Apps Script（受付窓口） → 先生のスプレッドシート
```

- 送信に失敗した記録（Wi-Fi切れなど）は生徒のPC内に残り、次にアプリを開いたとき自動で再送されます。
- 連携を設定しなくてもアプリは今まで通り動きます（記録は各PC内のみ）。

## 手順1：スプレッドシートを作る

1. [Google スプレッドシート](https://sheets.google.com) で新しいスプレッドシートを作成
2. 名前を「パソコンレッスン学習記録」などに変更

## 手順2：Apps Script を設置する

1. スプレッドシートのメニューから **拡張機能 → Apps Script** を開く
2. 最初から書いてあるコードを全部消して、下のコードを貼り付ける

```javascript
function doPost(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const data = JSON.parse(e.postData.contents);

  // 設定の更新（先生用ページの「まとめテスト」スイッチ）
  if (data.setting) {
    const settings = getSettingsSheet_(ss);
    settings.getRange("B2").setValue(data.setting.testEnabled === true);
    return ContentService.createTextOutput("ok");
  }

  let sheet = ss.getSheetByName("学習記録");
  if (!sheet) {
    sheet = ss.insertSheet("学習記録");
  }
  if (sheet.getLastRow() === 0) {
    sheet.appendRow(["日付", "時刻", "学籍番号", "名前", "レッスン", "記録", "詳細", "時間(秒)", "ミス回数"]);
  }

  const rows = (data.rows || []).map(function (r) {
    return [
      r.date,
      r.time,
      r.studentId,
      r.name || "",
      r.lesson,
      r.event,
      r.detail || "",
      r.timeSec === "" || r.timeSec == null ? "" : r.timeSec,
      r.missCount === "" || r.missCount == null ? "" : r.missCount,
    ];
  });

  if (rows.length > 0) {
    sheet.getRange(sheet.getLastRow() + 1, 1, rows.length, rows[0].length).setValues(rows);
  }

  return ContentService.createTextOutput("ok");
}

// アプリが設定（まとめテストの表示/非表示）を読みに来る
function doGet(e) {
  const ss = SpreadsheetApp.getActiveSpreadsheet();
  const settings = getSettingsSheet_(ss);
  const enabled = settings.getRange("B2").getValue() === true;
  return ContentService
    .createTextOutput(JSON.stringify({ testEnabled: enabled }))
    .setMimeType(ContentService.MimeType.JSON);
}

// 「設定」シートを取得（なければ作成）
function getSettingsSheet_(ss) {
  let sheet = ss.getSheetByName("設定");
  if (!sheet) {
    sheet = ss.insertSheet("設定");
    sheet.getRange("A1").setValue("設定項目");
    sheet.getRange("B1").setValue("値");
    sheet.getRange("A2").setValue("まとめテストを表示する");
    sheet.getRange("B2").setValue(false);
  }
  return sheet;
}
```

3. 💾（保存）を押す

## 手順3：Webアプリとして公開する

1. 右上の **デプロイ → 新しいデプロイ** をクリック
2. ⚙（歯車）→ **ウェブアプリ** を選ぶ
3. 次のように設定する
   - **次のユーザーとして実行**: 自分
   - **アクセスできるユーザー**: **全員**（←重要。生徒はGoogleにログインせずに送信するため）
4. **デプロイ** を押す（初回は「アクセスを承認」が出るので、自分のアカウントで許可する）
5. 表示された **ウェブアプリのURL**（`https://script.google.com/macros/s/…/exec`）をコピーする

> ⚠️ URLを知っている人はこのシートに行を追加できます。URLは生徒に教えず、シートの閲覧権限も共有しないでください。

## 手順4：Vercel にURLを設定する

1. [Vercel](https://vercel.com) のプロジェクト（windows-pc-lesson）を開く
2. **Settings → Environment Variables** を開く
3. 次の環境変数を追加する
   - **Name**: `NEXT_PUBLIC_SHEETS_WEBHOOK_URL`
   - **Value**: 手順3でコピーしたURL
   - **Environment**: Production（Preview / Development もお好みで）
4. **Deployments** タブから最新のデプロイの「…」→ **Redeploy** を実行（環境変数はビルド時に埋め込まれるため再デプロイが必要です）

## 確認方法

1. 本番URLでアプリを開き、学籍番号を入れてログイン
2. どれかのレッスンを開始する
3. スプレッドシートに「学習記録」シートができて、行が増えていればOK！
4. アプリの `/teacher` ページを開くと「✅ スプレッドシート連携：オン」と表示されます

## まとめテストの一括切り替えについて

この連携を設定すると、「まとめテスト」の表示/非表示が**スプレッドシートに保存**され、
先生用ページ（/teacher）のスイッチ1つで**教室のすべてのパソコンに反映**されます（各端末には20秒ほどで反映）。

- スイッチの状態はスプレッドシートの「**設定**」シート（自動で作られます）のB2セルに保存されます。
  B2セルを直接 `TRUE` / `FALSE` に書き換えても切り替えられます。
- 連携を設定していない場合は、スイッチはそのパソコンにしか反映されません。
- **以前のバージョンのコードを設置済みの場合**は、上のコードを貼り直して、下記の手順で再デプロイしてください。
  対応できていない間は、先生用ページに ⚠️ の案内が表示されます。

## コードを直したときは（Apps Script側）

Apps Script のコードを変更した場合は、**デプロイ → デプロイを管理 → ✏（編集）→ バージョン: 新バージョン → デプロイ** で更新してください（保存だけでは公開中のWebアプリに反映されません）。URLは変わりません。
